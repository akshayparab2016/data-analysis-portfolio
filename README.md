# ✨ Data Analysis Portfolio

Welcome to my **Data Analysis Portfolio** repository! This project serves as a showcase for practical data manipulation, exploration, and business intelligence projects. It highlights automated data workflows using Python, Pandas, and Excel integration to extract meaningful business insights from raw data.

---

## 📂 Repository Structure

This repository contains the following core data analytics projects:

### 1. Pandas Excel Project (`Pandas_excel_project`)
* **Objective:** Bridge the gap between spreadsheet management and programmatic data analysis by leveraging Python to automate Excel workflows.
* **Key Operations:** Loading multi-sheet workbooks, data cleaning, applying complex formatting, handling formulas programmatically, and aggregating large-scale datasets.
* **Core Toolkit:** Python, Pandas, OpenPyXL / XlsxWriter.

### 2. Python Sales Analysis Project (`Python_Sales_Analysis_Project`)
* **Objective:** Clean, explore, and analyze commercial sales transactions to identify key performance drivers, seasonal trends, and purchasing patterns.
* **Key Operations:** Handling missing data, computing Exploratory Data Analysis (EDA) metrics, calculating key performance indicators (KPIs) like revenue growth and profit margins, and generating data visualizations.
* **Core Toolkit:** Python, Pandas, NumPy, Matplotlib / Seaborn.

---

## 🛠️ Technical Stack

* **Data Manipulation:** `pandas`, `numpy`
* **Data Integration:** `openpyxl`, `xlrd` (for Excel engine handling)
* **Data Visualization:** `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebooks / Python Scripts
